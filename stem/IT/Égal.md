# INTRODUCTION

*Égal* (hereafter, Egal) is a decentralised, resilient, web-borne computing platform for deploying small-scale, in-house-developed business applications. Its name reflects the egalitarian principle that governs the relationship amongst the participating computers, which are all coequal hosts. That is, there is no hosting service provider who has dominion over the application servers, the data, and the users and, as such, there are no contractual commitments, no hosting fees. In essence, Egal is a peer-to-peer micro computing platform where data are held in an encrypted, distributed ledger held by a small number of hosts with shared, equal responsibilities.

Every participating host is just a web browser (or a Node server) running on a user's device, be it a server, a desktop, a laptop, a tablet, or a phone. Any host can start up the Egal web application to join the peer-to-peer network as a coequal participant.
